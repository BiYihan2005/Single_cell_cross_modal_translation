# 方法说明

## 1. 任务定义

本项目研究的是**单细胞跨模态预测问题**：给定一个细胞的 DNA 甲基化特征，预测该细胞对应的 RNA 表达向量。也就是说，模型需要学习从甲基化模态到 RNA 表达模态的映射关系：

```math
X^{\mathrm{met}}_i
\longrightarrow
Y^{\mathrm{rna}}_i
```

其中，X^{\mathrm{met}}_i 表示第 $i$ 个细胞的甲基化特征，Y^{\mathrm{rna}}_i 表示第 $i$ 个细胞对应的 RNA 表达向量。

该任务的难点主要包括：

1. 输入和输出来自不同生物模态，二者不是简单的一一对应关系；
2. RNA 表达矩阵通常维度很高，直接预测完整表达向量难度较大；
3. 单细胞数据噪声较强，测序深度、稀疏性和技术误差都会影响建模效果；
4. 细胞类型差异往往是 RNA 表达差异的主导来源，因此模型需要充分利用细胞类型结构信息。

## 2. 两阶段建模思想

本项目没有直接训练一个端到端神经网络去预测完整 RNA 表达矩阵，而是将任务拆解为三个部分：

1. **细胞类型软分类**：根据甲基化特征估计测试细胞属于各个 MajorType 的概率；
2. **细胞类型 RNA 原型表达**：计算训练集中每个 MajorType 的平均 RNA 表达，作为该类型的原型表达；
3. **残差神经网络修正**：用神经网络学习原型表达无法解释的个体差异。

最终预测公式为：

```math
\widehat{Y}_{\mathrm{test}}
=
P_{\mathrm{test}} C
+
\alpha \cdot g_{\theta}(Z_{\mathrm{test}})
```

其中，$P_{\mathrm{test}}C$ 是基于细胞类型概率和原型表达得到的主预测，$g_{\theta}(Z_{\mathrm{test}})$ 是残差神经网络预测的修正项，$\alpha$ 是残差融合权重。

各符号含义如下：

* $P_{\mathrm{test}}$：测试细胞属于各个 MajorType 的概率矩阵；
* $C$：训练集中各个 MajorType 的平均 RNA 表达原型；
* $Z_{\mathrm{test}}$：测试甲基化数据经过预处理和 PCA 后得到的低维表示；
* $g_{\theta}$：残差神经网络，用于学习原型表达无法解释的细胞个体差异；
* $\alpha$：残差融合权重，默认取 0.4。

直观来说，模型先根据 DNA 甲基化特征判断测试细胞更像哪些细胞类型，再用这些细胞类型的 RNA 表达原型生成基础预测，最后用残差神经网络对基础预测进行修正。

## 3. 为什么使用软分类

硬分类只给出一个确定的类别标签，例如“该细胞属于 Type A”。但在单细胞数据中，细胞状态可能存在连续过渡，某些细胞也可能处于分化过程中的中间状态。因此，单一类别标签可能会丢失部分信息。

软分类则输出一个概率向量，例如：

```text
Type A: 0.70
Type B: 0.20
Type C: 0.10
```

这表示该细胞最接近 Type A，但也保留了一定的 Type B 和 Type C 信息。相比硬分类，软分类能够更细致地表达细胞类型的不确定性和连续过渡状态。

在本项目中，软分类概率用于对不同 MajorType 的 RNA 原型表达进行加权平均，从而得到测试细胞的初步 RNA 表达预测。

## 4. 原型表达

对于每一个 MajorType，本项目计算训练集中属于该类型的细胞的平均 RNA 表达，作为该类型的 RNA 原型表达：

```math
C_k
=
\frac{1}{|S_k|}
\sum_{i \in S_k} Y_i
```

其中，$S_k$ 表示第 $k$ 个细胞类型的训练细胞集合，$C_k$ 表示第 $k$ 个细胞类型的平均 RNA 表达原型。

对于一个测试细胞，如果它属于不同 MajorType 的概率为 $P_{\mathrm{test}}$，则其初步 RNA 预测可以表示为各类型原型表达的加权平均：

```math
\widehat{Y}^{\mathrm{proto}}_{\mathrm{test}}
=
P_{\mathrm{test}}C
```

这种做法的好处是，模型首先利用了细胞类型这一强结构信息。由于细胞类型往往决定了 RNA 表达的大部分差异，原型表达可以提供一个较稳健的基础预测。

## 5. 残差学习

仅使用细胞类型原型表达仍然无法解释同一细胞类型内部的个体差异。因此，本项目进一步引入残差神经网络，用来学习原型表达无法解释的剩余部分。

训练集中，原型预测与真实 RNA 表达之间的残差定义为：

```math
R_i
=
Y_i
-
\widehat{Y}^{\mathrm{proto}}_i
```

残差网络的目标是学习：

```math
g_{\theta}(Z_i)
\approx
R_i
```

其中，$Z_i$ 是第 $i$ 个细胞的甲基化低维表示，$g_{\theta}$ 是神经网络。

这样设计的好处是：神经网络不需要从零开始预测完整 RNA 表达矩阵，而只需要学习原型表达无法解释的差异。相比直接端到端预测，这种方式降低了训练难度，也提高了模型的可解释性。

## 6. 后处理

RNA 表达量通常具有非负性，因此最终预测结果需要进行非负截断。具体而言，如果模型预测出负值，则将其截断为 0：

```math
\widehat{Y}
=
\max(\widehat{Y}, 0)
```

这一处理可以保证最终输出结果符合 RNA 表达数据的基本生物学含义。

## 7. 可进一步改进方向

后续可以从以下几个方向继续改进：

* 使用更适合稀疏矩阵的大规模训练方式；
* 引入 gene-level biological prior，使模型更好地利用基因层面的先验知识；
* 使用自编码器或变分模型学习甲基化模态和 RNA 模态之间的共享潜空间；
* 使用图神经网络建模细胞之间的邻域关系；
* 对残差融合权重 $\alpha$ 做验证集搜索，而不是固定为 0.4；
* 增加更严格的训练集、验证集和测试集划分协议，提高模型评估的可靠性。
