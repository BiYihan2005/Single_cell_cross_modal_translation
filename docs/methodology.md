# 方法说明

## 1. 任务定义

目标是学习从甲基化模态到 RNA 表达模态的映射：

\[
X^{met}_i \longrightarrow Y^{rna}_i
\]

其中 \(X^{met}_i\) 表示第 \(i\) 个细胞的甲基化特征，\(Y^{rna}_i\) 表示对应 RNA 表达向量。

该任务难点在于：

1. 输入和输出是不同生物模态；
2. RNA 输出维度高；
3. 单细胞数据噪声强；
4. 细胞类型差异是表达差异的主导来源。

## 2. 两阶段建模思想

本项目没有直接训练一个端到端网络，而是把任务拆成：

1. 细胞类型软分类；
2. 细胞类型 RNA 原型表达；
3. 残差神经网络修正。

最终预测为：

\[
\widehat{Y}_{test}
=
P_{test} C
+
\alpha g_{\theta}(Z_{test})
\]

其中 \(P_{test}C\) 是基于细胞类型概率和原型表达得到的主预测，\(g_{\theta}(Z_{test})\) 是残差网络预测的修正项。

## 3. 为什么使用软分类

硬分类只给出一个类型标签，例如“该细胞属于 Type A”。软分类给出概率向量，例如：

```text
Type A: 0.70
Type B: 0.20
Type C: 0.10
```

单细胞状态可能存在连续过渡，因此概率向量比单一标签保留更多信息。

## 4. 原型表达

对每个 MajorType 计算平均 RNA 表达：

\[
C_k = \frac{1}{|S_k|}\sum_{i\in S_k} Y_i
\]

其中 \(S_k\) 表示第 \(k\) 个细胞类型的训练细胞集合。测试细胞的初步 RNA 预测为各类型原型的加权平均。

## 5. 残差学习

训练集中原型预测与真实 RNA 的差为：

\[
R_i = Y_i - \widehat{Y}^{proto}_i
\]

残差网络学习：

\[
g_{\theta}(Z_i) \approx R_i
\]

这样网络不需要从零开始预测完整表达矩阵，而只需要学习原型无法解释的差异，训练难度更低。

## 6. 后处理

RNA 表达量通常非负，因此最终预测后将负值截断为 0：

\[
\widehat{Y} = \max(\widehat{Y}, 0)
\]

## 7. 可进一步改进方向

- 使用更适合稀疏矩阵的大规模训练方式；
- 引入 gene-level biological prior；
- 使用自编码器或变分模型学习共享潜空间；
- 使用图神经网络建模细胞邻域关系；
- 对残差权重 \(\alpha\) 做验证集搜索；
- 增加更严格的训练/验证评估协议。
