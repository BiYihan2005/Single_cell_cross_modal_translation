# 项目摘要

本项目研究单细胞多组学中的跨模态翻译问题：如何利用 DNA 甲基化特征预测 RNA 表达。相比直接端到端回归，本项目采用“细胞类型原型表达 + 残差神经网络”的分解式建模策略。

## 核心贡献

1. 将高维跨模态预测任务拆解为更可解释的两阶段流程；
2. 使用 MajorType 软分类概率构建 RNA 原型表达；
3. 使用残差神经网络学习原型无法解释的个体差异；
4. 提供可复用命令行 pipeline 和合成数据 demo；

## 方法优势

- 原型表达提供稳定的生物学先验；
- 软分类概率保留细胞状态连续性；
- 残差网络避免从零学习完整高维输出；
- 残差融合权重降低过拟合噪声的风险。

## 适用场景

该方法适合用于教学型、实验型或入门研究型的单细胞跨模态预测任务。对于大规模生产环境，仍需进一步优化内存管理、稀疏矩阵处理和模型验证方案。
