# Data Format

This repository does not include the original course or benchmark dataset. Raw single-cell data may have redistribution restrictions, and large `.h5ad` files are not suitable for a lightweight public GitHub repository.

## Required Files

The pipeline expects three AnnData files:

```text
data/raw/adata_rna_train.h5ad
data/raw/adata_met_train.h5ad
data/raw/adata_met_test.h5ad
```

## 1. RNA Training Data

`adata_rna_train.h5ad`

- Rows: training cells.
- Columns: RNA genes.
- `obs_names`: cell identifiers.
- `var_names`: gene identifiers or gene names.
- `X`: RNA expression matrix.

## 2. Methylation Training Data

`adata_met_train.h5ad`

- Rows: training cells.
- Columns: methylation features.
- `obs_names`: cell identifiers.
- `var_names`: methylation feature identifiers.
- `obs["MajorType"]`: major cell-type labels used for prototype construction.

The label column can be changed with:

```bash
--major_type_col YOUR_COLUMN_NAME
```

## 3. Methylation Test Data

`adata_met_test.h5ad`

- Rows: test cells.
- Columns: methylation features.
- `obs_names`: cell identifiers.
- RNA labels are not required.

## Alignment Rules

The training RNA and methylation matrices are aligned by shared cell identifiers:

$$
\begin{aligned}
\mathcal{I}_{\mathrm{train}}
&=
\mathrm{obs}\!\left(X^{\mathrm{met}}_{\mathrm{train}}\right)
\cap
\mathrm{obs}\!\left(Y^{\mathrm{rna}}_{\mathrm{train}}\right).
\end{aligned}
$$

Only cells in this intersection are used for supervised training. Methylation features are also aligned between training and test methylation matrices by shared `var_names`.

## Synthetic Demo Data

A small synthetic dataset can be generated by:

```bash
python scripts/make_synthetic_h5ad.py --output_dir data/synthetic
```

The synthetic data are only intended to verify that the pipeline runs end to end. They are not intended to represent realistic biological mechanisms.
